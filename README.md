# VocabBoost

## Description
VocabBoost is a vocabulary learning application designed to help users improve thei English vocabulary skills through studying, quizzes, and progress tracking.

## Problem the App Solves
Many students and English learners struggle to remember vocabulary words and stay consistent with studying. VocabBoost helps users learn new words in an organized and interactive way.

## Target users
- Students
- English learners
- People preparing for exams
- Anyone wanting to improve vocabulary skills

## Key Features
- User login and account creation
- Vocabulary study section
- Multiple choice quizzes
- Progress tracking
- Vocabulary level selection
- Review missed words
- User settings

# How the application Works
Users create an account or log in to the app. They can choose a vocabulary level and study vocabulary words with meanings and example sentences. Users can also take quizzes to test their knowledge. The app tracks scores, progress, and learning statistics to help users improve over time.
